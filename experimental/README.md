# Experimental Modules

These modules contain experimental optimization approaches that are **not verified** and **not integrated** into the core system.

They are preserved for reference and potential future development.

| Module | Description | Status |
|--------|-------------|--------|
| `compressed_sensing/` | CSWC witness compression | Untested |
| `tropical/` | Tropical geometry for MaxPool/Softmax | Untested |
| `wavelet/` | Haar wavelet witness batching | Untested |
| `deployment/` | API server + CLI scaffolding | Stub |
| `contracts/` | Solidity contract templates | Empty |
